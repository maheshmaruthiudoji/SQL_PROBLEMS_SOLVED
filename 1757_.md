#  LeetCode 1757: Recyclable and Low Fat Products

## Table: Products

| Column Name | Type           |
| ----------- | -------------- |
| product_id  | int            |
| low_fats    | enum ('Y','N') |
| recyclable  | enum ('Y','N') |

* `product_id` is the primary key
* `low_fats = 'Y'` → low fat
* `recyclable = 'Y'` → recyclable

---

## Problem

Write an SQL query to find the **product IDs** that are both **low fat** and **recyclable**.

Return the result in any order.

---

## Solution

```sql id="8o98aa"
SELECT product_id
FROM Products
WHERE low_fats = 'Y' AND recyclable = 'Y';
```
