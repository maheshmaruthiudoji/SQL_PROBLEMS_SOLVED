# LeetCode 584: Find Customer Referee

##  Table: Customer

| Column Name | Type    |
| ----------- | ------- |
| id          | int     |
| name        | varchar |
| referee_id  | int     |

* `id` is the primary key
* `referee_id` refers to another customer's id

---

## Problem

Find the **names of customers** who:

* are **not referred by customer with id = 2**, OR
* **have no referee (NULL)**

Return the result in any order.

---

##  Solution

```sql id="k29x7a"
SELECT name
FROM Customer
WHERE referee_id != 2 OR referee_id IS NULL;
```
